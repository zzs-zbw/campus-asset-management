# 校园实训楼固定资产资产管理系统 - 运行教程

## 一、环境准备

### 1.1 安装JDK 8

1. 下载JDK 8：https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html
2. 安装JDK并配置环境变量：
   - 新建系统变量 `JAVA_HOME`，值为JDK安装路径
   - 在 `Path` 变量中添加 `%JAVA_HOME%\bin`
3. 验证安装：打开命令行输入 `java -version`

### 1.2 安装MySQL 8.0

1. 下载MySQL：https://dev.mysql.com/downloads/mysql/
2. 安装MySQL并设置root密码
3. 启动MySQL服务
4. 验证安装：打开命令行输入 `mysql -u root -p`

### 1.3 安装Maven 3.6

1. 下载Maven：https://maven.apache.org/download.cgi
2. 解压到指定目录
3. 配置环境变量：
   - 新建系统变量 `MAVEN_HOME`，值为Maven解压路径
   - 在 `Path` 变量中添加 `%MAVEN_HOME%\bin`
4. 验证安装：打开命令行输入 `mvn -version`

### 1.4 安装Node.js 16+

1. 下载Node.js：https://nodejs.org/
2. 安装Node.js（npm会自动安装）
3. 验证安装：打开命令行输入 `node -v` 和 `npm -v`

## 二、数据库配置

### 2.1 导入数据库

1. 打开MySQL命令行或使用Navicat等工具
2. 执行以下命令导入数据库：

```bash
mysql -u root -p < asset_management.sql
```

或在MySQL命令行中：

```sql
source d:\桌面\零代码\asset_management.sql
```

### 2.2 验证数据库

1. 登录MySQL：`mysql -u root -p`
2. 查看数据库：`SHOW DATABASES;`
3. 使用数据库：`USE asset_management;`
4. 查看表：`SHOW TABLES;`
5. 查看初始数据：`SELECT * FROM sys_user;`

## 三、后端启动

### 3.1 修改配置文件

1. 打开文件：`backend/src/main/resources/application.yml`
2. 修改数据库连接信息：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/asset_management?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: 你的MySQL密码
```

### 3.2 编译项目

1. 打开命令行，进入后端目录：

```bash
cd d:\桌面\零代码\backend
```

2. 执行Maven编译：

```bash
mvn clean install
```

等待编译完成，看到 `BUILD SUCCESS` 表示成功。

### 3.3 启动后端服务

方式一：使用Maven启动

```bash
mvn spring-boot:run
```

方式二：使用IDE启动

1. 使用IntelliJ IDEA或Eclipse打开项目
2. 找到主类 `AssetManagementApplication.java`
3. 右键选择 `Run` 或 `Debug`

### 3.4 验证后端启动

看到以下信息表示启动成功：

```
========================================
校园实训楼固定资产资产管理系统启动成功！
访问地址：http://localhost:8080/api
========================================
```

访问 http://localhost:8080/api/druid 可以查看数据库连接池监控（账号：admin/admin）

## 六、常见问题解决

### 6.1 端口被占用

**后端端口8080被占用：**

修改 `application.yml` 中的端口号：

```yaml
server:
  port: 8081
```

**前端端口3000被占用：**

修改 `vite.config.js` 中的端口号：

```javascript
server: {
  port: 3001,
  // ...
}
```

### 6.2 数据库连接失败

1. 检查MySQL服务是否启动
2. 检查用户名和密码是否正确
3. 检查数据库是否已创建
4. 检查防火墙设置

### 6.3 Maven依赖下载失败

1. 配置Maven镜像源（阿里云）：

编辑 `settings.xml`：

```xml
<mirrors>
  <mirror>
    <id>aliyunmaven</id>
    <mirrorOf>*</mirrorOf>
    <name>阿里云公共仓库</name>
    <url>https://maven.aliyun.com/repository/public</url>
  </mirror>
</mirrors>
```

2. 清理Maven缓存：

```bash
mvn clean
```

### 6.4 npm依赖安装失败

1. 使用淘宝镜像源：

```bash
npm config set registry https://registry.npmmirror.com
```

2. 清理缓存重新安装：

```bash
npm cache clean --force
npm install
```

### 6.5 前端页面空白

1. 检查后端服务是否正常启动
2. 检查浏览器控制台是否有错误
3. 清除浏览器缓存
4. 检查API代理配置是否正确

## 七、生产环境部署

### 7.1 后端打包部署

1. 打包项目：

```bash
cd backend
mvn clean package -DskipTests
```

2. 运行jar包：

```bash
java -jar target/asset-management-1.0.0.jar
```

3. 后台运行（Linux）：

```bash
nohup java -jar target/asset-management-1.0.0.jar > app.log 2>&1 &
```

### 7.2 前端打包部署

1. 构建生产版本：

```bash
cd frontend
npm run build
```

2. 将 `dist` 目录部署到Nginx：

```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    location / {
        root /path/to/dist;
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://localhost:8080/api;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```
